# Utsav-Thakur.github.io

My personal developer portfolio and interactive GitHub activity dashboard.

🚀 **Live Portfolio**: [https://utsav-thakur.github.io](https://utsav-thakur.github.io)

---

## ✨ Features
- **Dynamic GitHub API Integration**: Fetches public repositories count, follower metrics, and profile bio dynamically.
- **Top Languages Stacked Chart**: Computes total language bytes from repositories and renders a horizontal segmented stacked progress bar with an interactive legend.
- **TSP Metaheuristics Sandbox**: Features a canvas-based interactive simulation running a 2-Opt Traveling Salesperson Problem (TSP) solver.
- **GitHub Dark UI Style**: Styled with exact GitHub dark theme palettes, glassmorphism cards, and pulsing activity states.
- **Contribution Heatmap**: Generates a dynamic SVG tracking weekday-weighted commit values across 53 weeks with custom tooltips.
- **Offline Sync & Cache**: Uses local storage cache to handle API rate limiting.

---

## 🛠️ Tech Stack
- **Structure & Layout**: Semantic HTML5, CSS Flexbox & CSS Grid
- **Styling & Aesthetics**: Vanilla CSS with custom scrollbars, glowing borders, and keyframe animations
- **Logic & Visuals**: Vanilla JS, HTML5 Canvas API, SVG renderers, and Intersection Observers

---

## 🚀 Deployment Instructions

To run this dashboard locally:
1. Clone the repository.
2. Open `index.html` in your browser.
3. (Optional) Run a local HTTP server:
   ```bash
   python -m http.server 8000
   ```
   And visit `http://localhost:8000`.
